The language for building web pages.
HTML and CSS
  • HTML (Hypertext Markup Language) is the standard markup language used for creating the structure and content of web pages.
  • It provides a way to organize and present information on the web.
  • HTML uses tags to define the different elements of a webpage, such as headings, paragraphs, images, links, tables, and more. These tags surround the content and provide a structure that web browsers can understand and render appropriately.
  • CSS (Cascading Style Sheets) is a stylesheet language used for styling the appearance of HTML elements.
  • It allows web developers to control the visual presentation, layout, and design of a webpage.
  • CSS works by selecting HTML elements using selectors and applying specific rules or properties to those elements.
  • With CSS, you can modify colors, fonts, sizes, margins, padding, positioning, and other visual aspects to create a visually appealing and consistent design across web pages.
